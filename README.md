# Customer Churn Early Warning System

## 📌 Overview
This project predicts customer churn using machine learning and identifies key factors affecting customer retention.

---

## 🛠 Tech Stack
- Python (Pandas, NumPy)
- Machine Learning (Random Forest)
- Visualization (Matplotlib, Seaborn)

---

## 📊 Key Insights
- Month-to-month customers have higher churn
- High monthly charges increase churn risk
- New customers are more likely to leave
